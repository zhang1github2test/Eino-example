# 《我以凡躯载诸界》核心设定集

## 一、世界观底层逻辑

### 1. 创世真相

*   **创世神“盘”**：宇宙初开时的唯一真神。死后身体化作万物（山川、河流、法则）。
*   **天道（旧）**：并非公正无私，而是“盘”死后产生的**尸毒/癌变意志**。它为了维持自身存在，需要定期收割“变数”，防止有人超越它。
*   **万古神藏**：是“盘”残留的**纯净意识碎片**，加上 9999 代试图逆天者死后不甘的意志聚合体。它是天道的“漏洞”，也是新天道的种子。
*   **核心冲突**：天道要抹杀所有“漏洞”（神藏持有者），主角要修补神藏，推翻旧天道，建立新秩序。

### 2. 地图层级（对应 3000 章节奏）

*   **凡界（1-500 章）**：玄灵宗 → 大周王朝 → 东荒九州。侧重肉身修炼，神藏修复底座。
*   **仙界（501-1200 章）**：三大仙域 → 远古禁区。侧重神通法则，神藏点亮名讳。
*   **神界/混沌（1201-2200 章）**：外宇宙 → 混沌海 → 神陨之地。侧重规则权柄，收集“盘”之碎片。
*   **道界/本源（2201-3000 章）**：时光长河 → 创世废墟 → 维度之外。侧重因果改写，墓碑翻转。

## 二、金手指：万古神藏（核心设定）

### 1. 外在形态

*   **本体**：一块悬浮在主角识海中的**残破石碑**（墓碑）。
*   **碑文**：碑面上密密麻麻刻着**9999 个名字**（前代持有者）。大部分灰暗（死亡），少数微光（传承未取），主角是第 10000 个空白处。
*   **状态**：初始为破碎状态，只剩底座。随着吞噬灾难，裂纹愈合，碑身逐渐完整。

### 2. 核心机制（去系统化处理）

*   **严禁出现**：【系统提示】、【叮】、【数据面板】、【加点】。
*   **表现形式**：
    *   **反馈**：碑文震颤、亡魂低语、裂纹愈合、血字浮现。
    *   **声音**：无数嘈杂的低语声，或特定某代持有者的清晰意志（如暴躁老哥、悲情剑仙）。
    *   **能量**：**道痕**（灾难残留）、**因果**（命运线）、**意志**（传承）。
*   **变强逻辑**：
    1.  **吞噬灾难**：雷劫、诅咒、剧毒、刑罚、杀局。灾难越重，碑修复越快，主角肉身越强。
    2.  **点亮名讳**：完成碑上某代持有者的“遗愿”，解锁其最强技能/功法。
    3.  **收集碎片**：寻找散落在世间的“盘祖碎片”，修复碑首，获得规则权柄。
    4.  **墓碑翻转**：后期碑身完全修复后，墓碑翻转，背面显露“创世真文”，可重写世界规则。

### 3. 代价与限制

*   **代价**：并非扣除寿命（避免压抑），而是**承受痛苦**。吞噬灾难时，主角需亲身承受部分痛苦（痛觉可随碑修复而削弱）。
*   **限制**：碑有承载上限。若吞噬过快，主角意识可能被 9999 代意志淹没（走火入魔风险）。
*   **天弃体质**：主角自带吸引灾难属性。走路摔坑、喝水塞牙。前期是副作用，后期是“自助餐”。

## 三、人物设定

### 1. 主角：陆仁

*   **身份**：玄灵宗外门弟子 → 神藏第 10000 任持有者 → 新天道奠基人。
*   **性格**：
    *   **狠辣理性**：不圣母，有仇必报，利益至上。
    *   **坚韧自嘲**：面对倒霉命运，不哭诉，而是利用倒霉。口头禅：“贼老天，你就这点本事？”
    *   **底层社畜感**：前期为了两块灵石拼命，后期为了众生生存拼命，始终保留“过日子”的务实感。
*   **金手指互动**：把碑里的意志当“老爷爷”团，有时合作，有时互骂，有时交易。
*   **修炼风格**：以伤换伤，以灾养身。别人避劫，他迎劫。

### 2. 关键意志（碑中群像）

*   **第 9998 代（暴躁老哥）**：前期主要引导者。性格火爆，护短，喜欢骂主角弱，但关键时刻最可靠。
*   **第 8888 代（狂刀·厉天）**：第一个点亮的名讳。刀道极致，遗愿是斩断锁龙链。
*   **第 1 代（开创者）**：后期出现。神秘，知晓创世真相，意志最微弱但最核心。
*   **群魂**：平时嘈杂低语，战斗时可集体咆哮，提供爆发力。

### 3. 主要敌人

*   **天道清道夫**：专门猎杀神藏持有者的组织。遍布各大势力，伪装成普通修士。
*   **天道代言人**：各境界的顶尖强者（仙帝、神皇），被天道赐福，视持有者为病毒。
*   **旧天道本体**：最终 BOSS，“盘”的腐化意识。

### 4. 重要配角

*   **女配角（建议 1-2 位）**：不花瓶，有独立事业线。例如某位发现主角秘密但选择合作的圣女，或同样被天道压迫的异族公主。
*   **兄弟/盟友**：少数知晓真相的人，后期反抗军骨干。

## 四、力量体系（九重天制）

*为了支撑 3000 章，每大境分 9 小阶，突破需渡劫（主角的食物）。*

1.  **凡境**（1-500 章）：淬体→通脉→凝神→化海→洞玄→涅槃→生死→轮回→渡劫
2.  **仙境**（501-1200 章）：人仙→地仙→天仙→真仙→玄仙→金仙→太乙→大罗→仙帝
3.  **神境**（1201-2200 章）：半神→真神→主神→至高→神帝→神尊→神王→神皇→创世
4.  **道境**（2201-3000 章）：合道→证道→大道→无上→永恒

*注：主角升级通常伴随“吞噬大劫”，越级挑战是常态。*
